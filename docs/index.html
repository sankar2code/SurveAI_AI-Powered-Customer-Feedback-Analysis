About this mockup set

This is a self-contained, clickable prototype generated from Surve.AI — Product Requirements Document v2.0 (Team 10, MIT Mastering Design Thinking programme). It covers all five user flows (FR1–FR13) across four personas — product manager, CX director, market research analyst, and brand manager — plus the respondent-facing survey experience.

Every figure, name, theme and quote shown is fictional-but-plausible content drawn from the platform's domain (retail customer feedback) — not placeholder text — so layouts, density and edge cases can be evaluated realistically.
